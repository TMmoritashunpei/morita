アクアゾーン・プログラム紹介

■ アクアビクス（一般)

運動不足解消、腰痛改善、シェイプアップに効果的な水中エアロビクスです。水の抵抗と浮力を利用し、音楽に合わせて楽しく体を動かします。各関節の負担が少なく、泳げない方でも大丈夫です。

■  マタニティスイミング（妊婦の方対象)

健康的なマタニティライフのためのフィットネスプラン。専門家によるカウンセリングで目的別パーソナルプランを作成後、グループレッスンに入ります。

■  ベビースイミング（0歳児をもつ父母対象）

0歳児から水に慣れ親しむことにより、赤ちゃんの健やかな発育を目指すプランです。専用プール、2～3人のグループレッスンで赤ちゃんも安心お母さん、お父さんとのスキンシップを図りましょう。
