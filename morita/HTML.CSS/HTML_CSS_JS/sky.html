最終更新日：2018.4.10

「青空フリマ」は、一年に一度、中央公園で開催されているフリーマーケットです。
フリーマーケットとは、日本語で言えば「蚤の市」。不要になった洋服や子供のおもちゃ、雑貨など、ご家庭の不用品を持ち寄り、それを必要とする方にお安くお譲りする場所の事です。
業者の方でなくても、誰でも出店するj事ができます。
フリーマーケットは、むやみにごみを増やさないという点で、省資源やリサイクルにも役立っています。また、思わぬ掘り出し物があったり、売買を通していろいろな人と交流したりと、楽しみ方もたくさんあります。

2008年4月に始まった青空フリマも、第10回目を迎えることになりました。多くの方々のご利用・ご協力、ありがとうございます。今年も青空フリマをよろしくお願いいたします。さて、第10回青空フリマを下記の日程で開催いたします。今回から、出店スペースが1.5倍になりますので、車での出店も楽々！五山化お待ちしております。
